# AHSO CRM Blueprint

Tài liệu blueprint nghiệp vụ chi tiết hiện đang được giữ ở file nguồn gốc ngoài repo:

- `../AHSO-CRM-Blueprint.docx`

Trong phase khởi động này, repo đã được bootstrap theo `PROJECT_STRUCTURE.md` và bộ mockup trong thư mục `Design`. Khi chuẩn hóa lại tài liệu nghiệp vụ cho codebase, nội dung từ file `.docx` nên được chuyển dần sang Markdown trong thư mục `docs/`.
